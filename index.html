Let's get started

Congratulations! You decided on learning modern frontend techniques. You cannot imagine, how proud we are! Soon you will be building interative prototypes like a pro. And all the frontend developers around you will love you.

Step 1: Setup your computer

We suggest installing the following applications so we all have the same setup. That makes explaining much easier.

You should use a Mac. Yes, we know that a PC can be a great machine. But for modern frontend development, it is much easier on a Mac (or on Linux). Many programming languages, apps and libs have way better support and there are a few applications that are very helpful that only exist for machines running OS X (the operating system running on your Mac).

Install Google Chrome

First, we need a modern browser. The current best choice is Google Chrome. It is a very advanced, quick and robust browser. It has great built-in development tools, that are very helpful speeding up the development process.

Recommended extensions for Google Chrome

Responsive Inspector
Look how a website looks in the different browser widths defined by the developer
Bootstrap 3 Offline Guide
The full Bootstrap 3 documentation saved in your browser. No internet connection needed to stay productive!
Jetstrap
Graphical editor for Bootstrap. Not without flaws, but interesting enough to fiddle around a bit.

Install Sublime Text 3

There are a lot of great text editors out there. This one is very fast, very popular and there are a lot of great extensions. We will use a few of them. You can install the free (and endless) beta of Sublime Text 3 here.
